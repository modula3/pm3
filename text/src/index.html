Text

This section contains text and documentation processing libraries and tools.

SGML tools
Standard Generalized Markup Language (SGML) and HyperText Markup Language (HTML) based tools for documentation processing. These libraries and tools are used, among other things, for processing the documentation found in this Modula-3 distribution.
mtex
TeX like Markup language used to produce nroff, HTML and LaTeX documents. Only a portion of this package is available outside DEC SRC and its use is not recommended in PM3.
