RealFloat, LongRealFloat, and ExtendedFloat

The interfaces RealFloat, LongRealFloat, and ExtendedFloat define operations that depend on the floating-point representation. Each one is an instance of a generic interface Float: floating-point operations 


INTERFACE RealFloat = Float(Real) END RealFloat.
INTERFACE LongFloat = Float(LongReal) END LongFloat.
INTERFACE ExtendedFloat = Float(Extended) END ExtendedFloat.
