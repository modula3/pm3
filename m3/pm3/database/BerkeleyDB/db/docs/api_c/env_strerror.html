db_strerror

 

#include <db.h>


char *
db_strerror(int error);

Description

The db_strerror method returns an error message string corresponding to the error number error. This interface is a superset of the ANSI C X3.159-1989 (ANSI C) strerror(3) interface. If the error number error is greater than or equal to 0, then the string returned by the system interface strerror(3) is returned. If the error number is less than 0, an error string appropriate to the corresponding Berkeley DB library error is returned. See Error returns to applications for more information.
