Database statistics

The DB->stat method is the standard interface for obtaining database statistics. Generally, DB->stat returns a set of statistics about the underlying database, for example, the number of key/data pairs in the database, how the database was originally configured, and so on.

There is one flag you can set to customize the returned statistics:

DB_FAST_STAT
Return only information that can be acquired without traversing the entire database.
