Modula-3 Concise Bibliography

The complete annotated Modula-3 bibliography now has over 50 entries. This page is an attempt at a more concise list of essential Modula-3 references organized by topic. The links on this page are pointers to the annotated bibliography. Please send comments and/or additions to m3-request@src.dec.com.

Books

  • Systems Programming with Modula-3, Nelson, 1991.
  • Modula-3, Harbison, 1992.
  • Algorithms in Modula-3, Sedgewick, 1993.
  • Programming with Modula-3, Boeszoermenyi & Weich, 1995.

Overview Articles

  • Introducing Modula-3, Wyant, 1994.
  • A Happy User Promotes SRC Modula-3, Nayeri, 1996.
  • Modula-3, Harbison, 1992.
  • Safe Programming with Modula-3, Harbison, 1992.

Documentation of Modula-3 Interfaces

  • Some Useful Modula-3 Interfaces, Horning, Kalsow, McJones, & Nelson, 1993.
  • Network Objects, Birrell, Nelson, Owicki, & Wobber, 1994.
  • Trestle Reference Manual, Manasse & Nelson, 1991.
  • VBTkit Reference Manual: A toolkit for Trestle, Brown & Meehan.
  • The FormsVBT Reference Manual, Brown & Meehan.

Introductory Programming Articles

  • Modula-3 Reference and Tutorial, Schaub.
  • Net Balance: A Network Objects Example", Nayeri.
  • Building Distributed OO Applications: Modula-3 Objects at Work, Dagenais, 1995.
  • Partial Revelation and Modula-3, Freeman, 1995.
  • Initialization of Object Types, Nelson, 1995.
  • Trestle Tutorial, Manasse & Nelson, 1992.
  • Trestle by Example, Stansifer, 1994.

Modula-3 Ports

  • Using Modula-3 Under OS/2, Whimster, 1996.

Systems Written in Modula-3

  • The Juno-2 Constraint-Based Drawing Editor, Heydon & Nelson, 1994.
  • Zeus: A System for Algorithm Animation and Multi-View Editing, Brown, 1992.
  • Writing an Operating System with Modula-3,
    Sirer, Savage, Pardyak, DeFouw, & Bershad, 1995.
  • The Whole Program Optimizer, Diwan, 1995.

Parallel Programming

  • An Introduction to Programming with Threads, Birrell, 1989.
  • Synchronization Primitives for a Multiprocessor: A Formal Specification,
    Birrell, Guttag, Horning, & Levin, 1987.

Garbage Collection

  • Compacting Garbage Collection with Ambiguous Roots, Bartlett, 1988.
  • Distributed Garbage Collection for Network Objects,
    Birrell, Evers, Nelson, Owicki, & Wobber, 1993.

Comparisons to Other Languages

  • A Comparison of Modula-3 and Oberon-2, Boeszoermenyi, 1993.
  • A Comparison of Object-Oriented Programming in Four Modern Languages,
    Henderson and Zorn, 1993.
