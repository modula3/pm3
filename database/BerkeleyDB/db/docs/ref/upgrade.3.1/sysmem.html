Release 3.1: DB_SYSTEM_MEM

Using the DB_SYSTEM_MEM option on UNIX systems now requires the specification of a base system memory segment ID, using the DB_ENV->set_shm_key method. Any valid segment ID may be specified, for example, one returned by the UNIX ftok(3) interface.
