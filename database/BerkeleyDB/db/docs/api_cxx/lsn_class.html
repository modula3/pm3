DbLsn

 

#include <db_cxx.h>


class DbLsn { ... };

Description

The DbLsn object is a log sequence number which specifies a unique location in a log file. It has no methods and its data may not be manipulated by an application.

Class

DbEnv, DbLogc, DbLsn
