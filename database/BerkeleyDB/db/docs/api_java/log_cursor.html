DbEnv.log_cursor

 

import com.sleepycat.db.*;


public DbLogc log_cursor(int flags)
    throws DbException;

Description

The DbEnv.log_cursor method creates a log cursor.

The flags parameter is currently unused, and must be set to 0.

The DbEnv.log_cursor method throws an exception that encapsulates a non-zero error value on failure.

Errors

The DbEnv.log_cursor method may fail and throw an exception encapsulating a non-zero error for the following conditions:

EINVAL
An invalid flag value or parameter was specified.

The DbEnv.log_cursor method may fail and throw an exception for errors specified for other Berkeley DB and C library or system methods. If a catastrophic error has occurred, the DbEnv.log_cursor method may fail and throw a DbRunRecoveryException, in which case all subsequent Berkeley DB calls will fail in the same way.
