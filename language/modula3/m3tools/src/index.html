Modula-3 Programming tools

Tools for debugging, analysing and processing Modula-3 programs.

m3gdb
GDB based Modula-3 aware debugger.
gnuemacs
GNU emacs elisp files for editing Modula-3 source code.
m3ide
M3IDE a simple environment for M3 development under emacs.
pp
Modula-3 source code pretty printer.
coverage
Tool to analyze Modula-3 source code coverage data.
shownew
New memory allocations runtime monitoring graphical tool.
showthread
Thread runtime monitoring graphical tool.
m3browser
Modula-3 source code web based browser with full cross referencing of packages, modules and types.
m3bundle
Tool to bundle files (resources) with an executable.
m3tk
The Modula-3 source code analysis and transformation toolkit.
showheap
Program heap runtime monitoring graphical tool.
recordheap
Recorder for showheap.
replayheap
Replay recorded showheap files.
m3markup
Library to add HTML markup to Modula-3 source files.
m3scan
Scanner for Modula-3 source files.
m3tohtml
Convert a list of Modula-3 source files into a web of html files.
m3totex
Convert Modula-3 interfaces to formattable TeX files.
