Lectern bitmap documents viewer

The lectern system may be used to view and index documents only available in geometrical form (Postscript or bitmap).

lectern
Browse lectern documents.
buildlectern
Build lectern documents from Postscript or bitmap files.
editlectern
Edit the annotations associated with a lectern document.
lecternclient
Send a command to the lectern program.
lecterndoc
Internal format for lectern documents.
lecterntohtml
Create HTML indexes for directories containing lectern documents.
